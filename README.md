# jp-iptv-different-user
Reaperc's Japanese IPTV list with a VLC user agent in case you have issues. If you're signed into Github, you should be able to request that this repo update its lists to the latest version by running the [Sync IPTV submodule and patch M3U](https://github.com/Mvb1122/jp-iptv-different-user/actions/workflows/sync-m3u.yml) workflow. It should handle itself... but if it crashes hopefully I'll get an email and fix it.

# Use .m3u files in /out/
Main file can be downloaded directly [here](https://raw.githubusercontent.com/Mvb1122/jp-iptv-different-user/refs/heads/main/out/JP.m3u) or using the URL: https://raw.githubusercontent.com/Mvb1122/jp-iptv-different-user/refs/heads/main/out/JP.m3u

You can use the raw download URLs for the other ones.
